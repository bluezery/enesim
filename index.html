Introduction

Enesim is a rendering library that provides a complete solution for drawing, allocating, compositing and converting pixels. All of the abstractions of the library are flexible enough to support drawing capabilities on any scenario.

The most useful abstraction is the the Renderer system, which allows you to draw different graphics primitives (in Enesim it is called a Renderer) into a target drawable (in Enesim a Surface). Given that a Renderer can draw anything, it has its own state that is modified through the renderer's specific properties setters and getters.
